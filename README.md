### 使用步骤

- **下载脚本**
  - 在[release](https://gitee.com/zergqueen/Hearthstone-Script/releases)处下载
  - hs-script.zip包含完整依赖，一般下载此文件
  
- **初次使用**：
  - 配置卡组：游戏里将要使用的卡组移动到一号卡组位
  - 选择卡组：软件里选择要使用的卡组策略(先选择模式)
  
- **启动脚本**
  1. 双击 `hs-script.exe` 文件
  
  2. 点击开始按钮或者使用热键 `Ctrl` + `P`
  
- **关闭脚本**

  - 双击 `stop.bat` 文件
  - 使用热键 `Alt` + `P`  
  - 程序托盘处点击退出
